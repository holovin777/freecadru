# FreeCad - Как открыть контейнер?

1. Нажать на кнопку в углу экрана чтобы проверить, может быть уже включен контейнер

<img width="960" src="/Schermata%20del%202024-02-08%2009-33-50.png" alt="isolated" />

2. Если нет контейнера, пробуем поискать. Включаем программу File.

<img width="960" src="/Schermata%20del%202024-02-08%2010-50-42.png" alt="isolated" />

3. Нажимаем на Home.

<img width="960" src="/Schermata%20del%202024-02-08%2010-57-12.png" alt="isolated" />

4. В ней идем в папку GnomeLavoro -> Containers -> Manitou(На пример).
5. Открываем MTH192619-A-MA_MFG_MOD_3D-B1-stp.stp (Нужно открывать файл с форматом .stp).
6. Нажимаем на MTH192619_-_CONTENITORE_COMPLETO.

<img width="960" src="/Screenshot%20from%202024-02-08%2011-49-22.png" alt="isolated" />

7. "Fit all" или "View -> Standard views -> Fit all".

<img width="960" src="/Screenshot from 2024-02-09 13-55-27.png" alt="isolated" />
