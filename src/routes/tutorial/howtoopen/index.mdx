# FreeCad - Как открыть контейнер?

Нажать на кнопку в углу экрана чтобы проверить, может быть уже включен контейнер

<img width="1200" src="https://raw.githubusercontent.com/holovin777/freecadru/main/public/Schermata%20del%202024-02-08%2009-33-50.png" alt="isolated" />

Если нет контейнера, пробуем поискать. Включаем программу File.
Нажимаем на Home.
1. В ней идем в папку GnomeLavoro.
2. Containers.
3. На примере Manitou.
4. MTH192619-A-MA_MFG_MOD_3D-B1-stp.stp (Нужно открывать файл с форматом .stp).
