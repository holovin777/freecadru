# FreeCad - Как открыть контейнер?

1. Нажать на кнопку в углу экрана чтобы проверить, может быть уже включен контейнер

<img width="1200" src="https://raw.githubusercontent.com/holovin777/freecadru/main/public/Schermata%20del%202024-02-08%2009-33-50.png" alt="isolated" />

2. Если нет контейнера, пробуем поискать. Включаем программу File.

<img width="1200" src="https://raw.githubusercontent.com/holovin777/freecadru/main/public/Schermata%20del%202024-02-08%2010-50-42.png" alt="isolated" />

3. Нажимаем на Home.

<img width="1200" src="https://raw.githubusercontent.com/holovin777/freecadru/main/public/Schermata%20del%202024-02-08%2010-57-12.png" alt="isolated" />

4. В ней идем в папку GnomeLavoro.
5. Containers.
6. На примере Manitou.
7. Открываем MTH192619-A-MA_MFG_MOD_3D-B1-stp.stp (Нужно открывать файл с форматом .stp).
8. Нажимаем на MTH192619_-_CONTENITORE_COMPLETO
9. View - Standard views
